# Assets Overview

eRacun Validator requires external assets for validation:

- ISO Schematron engine
- Schematron rules (EN16931, PEPPOL, HR-CIUS)
- UBL XSD schemas
- Optional sample documents

Assets are **not bundled** with the package.

---

## Asset management modes

- Automatic download via CLI
- Manual offline installation
- Custom asset root (API / CLI)

---

## Default location

eracun_validator/asstes/
